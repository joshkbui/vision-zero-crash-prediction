# Models

This directory contains model artifacts for the Vision Zero Injury Risk Simulator.

## Trained Model

**XGBoost Crash Injury Classifier** (`xgb_model.pkl`)

| Property | Value |
|---|---|
| Algorithm | XGBoost (XGBClassifier) |
| Task | Binary Classification (Injury: 0/1) |
| n_estimators | 300 |
| max_depth | 6 |
| learning_rate | 0.05 |
| scale_pos_weight | 5 |
| Decision threshold | 0.40 |
| Injury Recall | **91%** |
| AUC-ROC | 0.92 |
| Weighted F1 | 0.84 |

## Generating the Model

The trained model is **not tracked in this repository** (excluded by `.gitignore`) due to its large binary size.

To generate the model artifact:

```bash
# 1. Download data (see data/README.md)
# 2. Open and run the training notebook
jupyter notebook notebooks/Final_Deployment_Code.ipynb
```

Running all cells will:
1. Load and preprocess the crash data
2. Apply SMOTETomek resampling
3. Train all four models (Logistic Regression, Random Forest, LightGBM, XGBoost)
4. Save the final XGBoost model to `models/xgb_model.pkl`
5. Generate SHAP plots to `images/`

## Resampling

SMOTETomek (combined oversampling + Tomek links cleaning) was applied to the training set only to address the ~60/40 class imbalance. The test set remains unmodified for honest evaluation.
